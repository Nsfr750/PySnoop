PySnoop Documentation
====================

Welcome to the PySnoop documentation! This documentation provides comprehensive information about the PySnoop application, including installation instructions, user guides, and developer documentation.

Building the Documentation
--------------------------

To build the documentation locally, follow these steps:

1. Install the required dependencies:

   .. code-block:: bash

       pip install -r requirements.txt

2. Build the HTML documentation:

   .. code-block:: bash

       make html

The documentation will be available in ``http://tuxxle.ddns.net/pysnoop/docs/``.

Documentation Structure
----------------------

- :doc:`installation` - Installation instructions
- :doc:`quickstart` - Quick start guide
- :doc:`user/interface` - User interface documentation
- :doc:`developer/api` - API reference
- :doc:`changelog` - Release history

Viewing the Documentation
------------------------

After building the documentation, open ``_build/html/index.html`` in your web browser to view the documentation.
