Features
========

- **Card Reading**: Support for various magnetic card readers
- **C10/Luhn Validation**: Built-in validation for card numbers
- **Database Management**: Store and manage card data
- **User-Friendly Interface**: Intuitive GUI for all operations
- **Cross-Platform**: Works on Windows, macOS, and Linux

Quick Start
-----------

1. Install PySnoop using pip::

    pip install pysnoop

2. Launch the application::

    pysnoop

3. Connect your card reader and start reading cards!

For more detailed instructions, see the :doc:`quickstart` guide.
